# 🔍 Job Vacancy Platform - Search Bar

A graphical job vacancy platform built in **C++ with WinBGI graphics library**. This application enables job seekers to search for jobs and employers to post/manage job listings through an interactive GUI with authentication and real-time search filtering.

**Live Job Search System** - Search desired items easily!

---

## 🎯 Project Overview

This project demonstrates a complete **job marketplace platform** with:
- 🔐 User authentication for employers and job seekers
- 🔍 Real-time search bar with live filtering
- 📋 Job posting and management system
- 👥 Role-based access control
- 💾 Persistent data storage with file I/O
- 🖱️ Interactive GUI with mouse/keyboard input
